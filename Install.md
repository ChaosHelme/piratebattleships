# Introduction #

Unpack the content in the Zip-file. Execute the Setup.exe order the Setup.msi -file. If ask for donwloading and installing the .Net Framework, click yes.

After downloading and installing the .Net Framework (only if it wasn't installed on your system) the Battleships pirate edition setup starts.

Choose your prefered install location and install the game. After the installer completed, a desktop shortcut will be created.

Double click the shortcut --> the game starts.
